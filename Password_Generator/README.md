# Random Password Generator

This project is developed for the **Oasis Infobyte Python Programming Internship**.

## Project Description

A Random Password Generator that creates secure passwords based on user preferences such as:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Symbols

## Features

- Strong password generation
- Customizable character sets
- Copy password button
- Modern UI design
- Python command-line version

## Technologies Used

HTML  
CSS  
JavaScript  
Python  

## How to Run

### Web Version

1. Download project files
2. Open `index.html` in browser

### Python Version (IDLE)

1. Open `password_generator.py`
2. Run the file
3. Enter password options
4. Generated password will appear
